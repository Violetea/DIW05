Resumen de como hacer un menú principal

TIPO DE ETIQUETAETIQUETACLASEDESCRIPCIÓN
Estructura del menú<nav>navbarIndica el navegador donde empieza un menú
<nav>navbar-expand-xl|lg|md|smIndica el tamaño de pantalla a partir del cual se colapsa el menú
<ul>navbar-navIndica donde empiezan los elementos del menú de navegación
<li>navbar-itemIndica cada uno de los elementos del menú de navegación
<a>navbar-linkIndica el elemento clickable dentro del menú de navegación
Aspecto visual<nav>navbar-dark|lightMuestra el menú de navegación del color indicado
<nav>bg-dark|lightEstablece el fondo de la barra de navegación al color indicado. Normalmente las clases navbar-dark y bg-dark suelen ir juntas
<a>navbar-brandNos permite incluir un logotipo ya sea una imagen o un texto en el menú de navegación
Posicionamiento<nav>fixed-topFija el menú de navegación arriba de la página web independientemente del lugar donde se tenga definido el menú.
<nav>fiexed-bottomFija el menú de navegación abajo de la página web independientemente del lugar donde se tenga definido el menú.
<nav>sticky-topEl menú se coloca donde se ha definido dentro del documento HTML pero al hacer scroll el menú se queda en la parte superior de la ventana.
Comportamiento responsive<button>navbar-toggler

Botón en el que podemos pulsar para extender el menú cuando este colapsado.

Para que se extienda el menú necesitamos definir los atributos data-toggle="collapse" y data-target="#target_name". Donde target name es el ID del div donde está navbar-nav

ATENCIÓN: no olvidar poner # delante de target_name

<span>navbar-toggler-iconEstablece el icono o el elemento que se mostrará cuando el menú está colapsado
<div>navbar-collapseEl DIV en el que se encuentran los elementos del menú debe pertenecer a esta clase. Además el ID del DIV debe coincidir con el introducido en target_name.
